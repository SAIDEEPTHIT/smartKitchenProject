# SmartKitchen – Leftovers to Recipes

SmartKitchen is a small MERN‑style web app that helps users turn leftover ingredients into recipes, manage a digital pantry, and save favourites.

## Main Features
- Login / Register (JWT auth)
- Pantry manager – add, edit, delete items
- Leftovers → Recipe search by ingredients
- Recipe details + simple grocery list
- Save and view favourite recipes

## Tech Stack
React (Vite) · Node.js · Express · MongoDB (Mongoose) · JWT · bcrypt

## Run Locally
<b>Backend</b>
cd backend
npm install
node server.js

Frontend
cd frontend
npm install
npm run dev
